The data for COVID deaths and vaccinations is analyzed in project. 
Use excel file [CovidVaccinations](https://github.com/rsgilltc/COVID/blob/main/CovidVaccinations.xlsx) and [CovidDeaths](https://github.com/rsgilltc/COVID/blob/main/CovidDeaths.xlsx) for the RAW data.
Power query and excel is used for data cleaning and transformation.
[Power BI](https://github.com/rsgilltc/COVID/blob/main/Covid.pdf) is used for visulatizations and [report](https://github.com/rsgilltc/COVID/blob/main/COVID%20January%202020%20to%20April%202021%20Report.pdf) for further detals.
